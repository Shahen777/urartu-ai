База: готовый сайт-ОС в этой папке (site-os/). ДНК эталона: ../taste/displace.agency.md. Все правила прежние: system-font, ноль внешних ресурсов, движение только transform/opacity, prefers-reduced-motion, тексты не менять (только переводить).

## Разделение владения файлами (для параллельной работы)

- **Агент «Ассеты»** владеет ТОЛЬКО: `wallpapers/*.svg`, `icons/*.svg` (новые и замена существующих). НЕ трогает html/css/js.


## 1. Обои (Ассеты) — «красивый фон, не чёрный»

Стиль macOS Sonoma/Sequoia — абстрактные мягкие волны из размытых градиентов, БЕЗ фиолетового доминирования:
- `dark.svg`: глубокий сине-чернильный низ → бирюзовые и тёплые янтарные всполохи-волны (аврора над горизонтом), общая тональность тёмная, чтобы белый текст читался. Размытие feGaussianBlur 60–90, 1600×1000.
- `light.svg`: пастель — небесно-голубой → мятный → тёплый персиковый, светлая, чтобы тёмный текст читался.
- Вес каждого ≤ 60KB (простые формы + blur). Подключение: CSS `background-image:url(...)` + `background-size:cover` + тёмный/светлый оверлей-градиент для контраста виджетов.

## 2. Папки и файлы «как у Apple» (Ассеты + Ядро)

- Иконка папки: классическая macOS-папка (передняя/задняя створка, градиент #4EA3F5→#2E7CD6, блик сверху), 512 viewBox.
- Файлы: белый лист с загнутым уголком, серые полоски текста; PDF-вариант с красным бейджем.
- Ядро: заменить иконки рабочего стола (Регламенты, База знаний → folder.svg; README.txt, Тарифы.txt → file-txt.svg) и в окнах-папках сделать Finder-подобный вид: заголовок-тулбар, список строк «иконка + имя + размер + дата», hover-подсветка строки, двойная колонка на десктопе.

## 3. Светлая и тёмная темы (Ядро)

- CSS-переменные двух тем. Тёмная = текущая. Светлая: стекло `rgba(255,255,255,0.62)` + blur, ring `rgba(0,0,0,0.08)`, текст `#1D1D1F` / `rgba(0,0,0,0.5)`, тени мягче (чёрный 0.08/0.18), меню-бар светлое стекло, окна `rgba(245,245,247,0.85)`, терминал остаётся тёмным всегда (он терминал).
- Переключатель: в меню-баре справа иконка солнце/луна (SVG). Порядок: авто (prefers-color-scheme) → клик переключает → сохранять в localStorage (`theme: dark|light|auto`— достаточно dark|light после первого клика).
- `<meta name="color-scheme" content="dark light">`. Переключение — класс `theme-light` на html, transition background 0.3s.
- Обои меняются вместе с темой (dark.svg/light.svg).

## 4. Английский язык (Ядро)

- Словарь переводов в `js/i18n.js` (новый файл, владеет Ядро): все строки интерфейса и контента окон, ВКЛЮЧАЯ диалоги чата-виджета и строки терминала. Перевод продающий, не машинный: заголовок «Your own AI. On your server. Data never leaves the company.»
- Переключатель в меню-баре: «РУ / EN» (как переключатель раскладки macOS). Механика: элементы с `data-i18n="key"`; переключение обновляет textContent; `<html lang>` меняется; выбор в localStorage; поддержка `?lang=en` в URL.
- SEO: `<link rel="alternate" hreflang="en" href="https://DOMAIN/?lang=en">` + hreflang ru + x-default; title/description тоже переключать.
- Цены оставить в ₽ в обеих версиях (рынок РФ), в EN добавить «(≈ from $1,700)» рядом с «от 150 000 ₽» один раз в тарифах.

## 5. Мобильная версия «как айфон» (Ядро)

- Статус-бар: время слева, справа глифы сигнала/Wi-Fi (перечёркнутый)/батареи (SVG), как iOS.
- Под виджетами — springboard: сетка иконок 4 колонки (все «приложения» + папки + файлы), подписи 11px, иконки 60px, отступы как на iPhone.
- Док iOS: полупрозрачная плашка со скруглением 32, 4 иконки, прижат к низу с safe-area (env(safe-area-inset-bottom)).
- Окна-листы: граббер-полоска сверху, slide-up 250ms cubic-bezier(0.32,0.72,0,1) (кривая iOS), свайп-вниз по грабберу закрывает (pointer events, порог 80px или velocity).
- Виджеты стопкой — уже есть, добавить внешний вид iOS 17: чуть крупнее радиус (22px), заголовок-ярлык с иконкой.
