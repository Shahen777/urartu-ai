# ⚠️ ВНИМАНИЕ: иконки — временные

Файлы `*.png` в этой папке — **системные иконки macOS**, извлечённые с компьютера
владельца для локальной демонстрации макета.

Они защищены авторским правом Apple Inc.

**Перед публикацией сайта в интернет их ОБЯЗАТЕЛЬНО заменить** на собственные
иконки. Генератор собственных иконок: `../tools/make_icons.py`
(запуск: `python3 tools/make_icons.py` — создаёт SVG в этой папке).

История: эталон displace.agency использует те же иконки Apple, поэтому его
рабочий стол выглядит «настоящим».
