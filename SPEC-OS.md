## Токены (строго из taste.json эталона)

- Обои: #000 с тёплым свечением ≈#1A100C радиально к центру-низу (два radial-gradient, без анимации)
- Стекло: rgba(24,24,28,0.55) + backdrop-filter blur(24px) + ring 0 0 0 1px rgba(255,255,255,0.10)
- Тени: виджеты `ring + 0 20px 25px -5px rgba(0,0,0,.1) + 0 8px 10px -6px rgba(0,0,0,.1)`; окна/док `0 12px 40px rgba(0,0,0,.35)`
- Текст: #F5F5F7 и rgba(255,255,255,0.45); акценты: красный #F3350C (точечно: метка дня, бейдж записи), жёлтый #FFD60A (один стикер)
- Радиусы: виджеты 18px, окна 16px, док pill, мелочь 8/6/4
- Шрифт: ТОЛЬКО `-apple-system, BlinkMacSystemFont, "Segoe UI", system-ui` — НИКАКИХ Google Fonts
- Шкала: 48/600 (цифры виджетов), 32/600, 16/400 (база), 13 (подписи иконок), 11/500 uppercase с разрядкой (ярлыки виджетов)
- Движение: transform/opacity 0.2s ease-out, ТОЛЬКО в ответ на действие (hover, открытие окна scale 0.96→1 + fade 200ms, закрытие 150ms). Исключение: печать текста в виджете ассистента. prefers-reduced-motion: всё мгновенно.

## Окна (оконный менеджер, js/os.js ~300 строк)

Окно: стекло темнее (rgba(24,24,28,0.85)), радиус 16px, title-bar с трафиклайтами (красный #FF5F57 закрыть, жёлтый #FEBC2E свернуть в док, зелёный #28C840 развернуть), draggable за title-bar (pointer events, capture), z-index менеджер (клик = наверх), позиции каскадом. ## Чего НЕ делать

Google Fonts, эмодзи-иконки, копирование PNG-иконок Apple, неоновое свечение, фиолетовые градиенты, автономные фоновые анимации (кроме печати чата и терминала), скролл-секции на десктопе, лоадеры-прелоадеры.
