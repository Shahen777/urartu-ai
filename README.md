# Urartu AI — сайт услуги внедрения локального ИИ

Сайт-визитка ООО «Урарту»: внедрение локального ИИ-ассистента, который отвечает по документам компании внутри её контура (соответствие 152-ФЗ).

**Живой сайт:** https://shahen777.github.io/urartu-ai/

## Идея

Сайт сделан как **симуляция операционной системы**: перед посетителем рабочий стол компании, на котором ИИ-ассистент уже установлен и отвечает по регламентам. Метафора сама доказывает продукт — перечёркнутый Wi-Fi в меню-баре, виджет «0 данных ушло наружу», терминал с выводом `данные наружу: 0 байт`.

## Что внутри

- Рабочий стол с виджетами и оконным менеджером (перетаскивание, изменение размера за 8 краёв, трафиклайты, магнификация дока)
- 11 окон-разделов: питч, ассистент, архитектура, отрасли, тарифы, вопросы, контакты, гарантии, терминал, папки
- **Звонок** прямо из браузера — аудио и видео через [Jitsi Meet](https://github.com/jitsi/jitsi-meet) (Apache-2.0)
- **3D-офис ИИ-агентов** — встроенный проект [The Delegation](https://github.com/arturitu/the-delegation) (MIT), русифицирован
- Тёмная и светлая темы, русский и английский языки
- Мобильная версия в виде iPhone: springboard, страницы со свайпом, док
- Ноль внешних ресурсов на первом экране: системный шрифт, локальные ассеты. Внешние запросы уходят только при явном действии (звонок, 3D-офис).

## Запуск локально

```bash
python3 -m http.server 8743
# открыть http://localhost:8743/
```

## Структура

```
index.html                      рабочий стол и все окна (SEO-разметка, JSON-LD)
politika-konfidencialnosti.html политика обработки ПДн (152-ФЗ)
css/os.css                      токены, темы, десктоп и iOS-режим
js/os.js                        оконный менеджер, док, виджеты, звонок
js/i18n.js                      словарь RU/EN
icons/                          иконки приложений
wallpapers/                     обои для тёмной и светлой тем
delegation/                     собранный 3D-офис агентов
tools/make_icons.py             генератор собственных иконок
```

## Лицензии и заимствования

- 3D-офис: [The Delegation](https://github.com/arturitu/the-delegation), MIT, автор Arturo Paracuellos
- Звонки: [Jitsi Meet](https://github.com/jitsi/jitsi-meet), Apache-2.0
- **Иконки приложений** (`icons/*.png`) — системные иконки macOS, использованы как временная заглушка макета. Права принадлежат Apple Inc. Подлежат замене на собственные (`tools/make_icons.py`). Подробнее: `icons/README.md`.
