Автор: Fable 5. Запускается ПОСЛЕ приёмки V5 (лоты делят файлы index.html/os.css/os.js/i18n.js).
Действуют правила BRIEF-V5.md («Порядок и правила», «Чего НЕ делать»), плюс: ни один новый модуль не должен грузить ничего внешнего при старте страницы; тяжёлое — только по клику.

## ЛОТ D — «Анимация окон + Калькулятор проекта» (модель: Fable — сложная механика)

### D1. Красивое открытие окон (глобально, деск+мобайл)
Сейчас: окно просто фейдится по центру — «некрасиво, нефункционально».
Требуется, как в настоящих macOS/iOS:
- **Десктоп:** окно вылетает ИЗ ИКОНКИ дока: при openWindow вычислить центр иконки-триггера, поставить transform-origin в эту точку (в координатах окна), анимировать scale(.25)→1 + opacity 0→1, 320ms cubic-bezier(.32,.72,0,1); при закрытии — обратно в иконку (240ms). Если триггер не из дока (меню-бар, desk-icon) — origin из него же. Reduced-motion: мгновенно.
- **Мобайл:** лист уже выезжает снизу; добавить старт из точки тапа по иконке springboard: короткая фаза scale(.4 от иконки)→полёт вверх в лист (можно упростить до: иконка пульсирует scale 1→1.15, экран затемняется, лист выезжает — главное ОЩУЩЕНИЕ отклика). Существующий app-open (фон scale .96) сохранить.
- Свернуть в док: окно «всасывается» в иконку (scale к origin иконки + fade).
- Проверить на WebKit-движке (тач) и Chrome.

### D2. Приложение «Калькулятор» — интеллектуальный расчёт проекта
Иконка: системный Calculator.app (icns → png/webp как остальные). Окно ~430px / мобильный лист.
Это НЕ арифметический калькулятор, а конфигуратор стоимости (как на urartu.pro, но красивее):
- Шаг 1 «Что делаем»: сетка 8 услуг (из окна Услуги V5; те же i18n-ключи цен): ИИ-сотрудник/внедрение, сайт, бот, голосовой ассистент, автоматизация, интеграция, видео, контент.
- Шаг 2 «Объём»: Бриф ×1 · Стандарт ×1.8 · Расширенный ×3 · Enterprise ×6 (сегмент-контрол iOS).
- Шаг 3 «Срочность»: Стандарт ×1 · Срочно ×1.3 (тумблер).
- Опции (чекбоксы): + Железо под ключ (ссылка на Устройства, добавляет «от 180 000 ₽»), + Поддержка (от 15 000 ₽/мес отдельной строкой).
- Живой итог: крупная сумма «от X ₽» пересчитывается с анимацией прокрутки цифр (tabular-nums, 300ms) при каждом изменении; под ней «≈ $Y» в EN-режиме наоборот.
- CTA: «Получить точную смету» → t.me/Shahen_kazaryan с prefilled-текстом расчёта (encodeURIComponent: услуга, объём, срочность, опции, итог) + вторая кнопка «Обсудить голосом» → окно Звонок.
- Дисклеймер: «Предварительная оценка. Финальная — после 30-минутного брифа.»
- Вся логика — чистый JS-объект тарифов; формулы прокомментировать.

## ЛОТ G — «Испытательный стенд: всё можно потрогать» (модель: Fable — ядро сложное)

Требование клиента: владелец бизнеса должен мочь ПОТЕСТИРОВАТЬ каждую функцию ИИ-сотрудников прямо на сайте, не «посмотреть сценарий», а поработать. Плюс добавить Фабрику контента.

### G1. Ядро `window.LocalAI` (js/localai.js, новый файл)
Трёхуровневая честная система, каждое приложение берёт лучший доступный уровень:
1. **WebLLM** (настоящая локальная модель в браузере): по явной кнопке «Запустить настоящий локальный ИИ (~индикатор объёма)» подгружается web-llm (модуль mlc-ai с CDN) + Qwen2.5-1.5B-Instruct q4 (WebGPU; на слабых — 0.5B). Прогресс загрузки — красивый экран «Устанавливаю модель…» с процентами. После загрузки — реальная генерация ОФЛАЙН-логики: это наш продукт в миниатюре, честный аргумент «модель работает у ВАС, данные не ушли — проверьте вкладку Network».
2. **BM25-поиск** (всегда доступен, мгновенный, ~80 строк): локальный индекс по встроенной демо-базе «регламенты юрфирмы „Вектор Права“» (тексты взять сжатые: регламент договорной работы — сроки согласования/суммы/утверждающие, положение о ПДн — сроки ответов/уведомление РКН 24ч/72ч, претензионный регламент, тарифы-биллинг; источник — наши старые demo-docs, воспроизвести компактно внутри js) + FAQ/услуги сайта. Ответ = найденный пункт с подсветкой и ссылкой-источником.
3. **Интенты** (из F1) — последний фолбэк и «личность» агентов.
ВАЖНО: внешние загрузки — ТОЛЬКО по явной кнопке пользователя с указанием объёма; при отказе/отсутствии WebGPU всё работает на уровнях 2–3. Правило «ноль внешнего при старте страницы» не нарушать.
Лот F проектирует чат сразу поверх интерфейса LocalAI.ask(agent, text) → {text, source?, chips?}.

### G2. Документовед — тест «на своих документах»
В чате Документоведа кнопка «Проверить на СВОЁМ документе»: панель — вставить текст (textarea) или выбрать .txt/.md файл → индексируется локально (BM25, уровень 1 если включён — суммаризация LLM) → пользователь задаёт вопросы по СВОЕМУ тексту, получает ответы с цитатой и подсветкой места. Плашка: «Файл не покидает ваш браузер. Это и есть принцип локального ИИ». Кнопка «Очистить».

### G3. Юрист-проверяющий — реальная проверка договора
Вставить текст договора → мгновенная детерминированная проверка по чек-листу (регексы/эвристики, ~15 правил из наших демо-регламентов: предмет, цена/НДС, неустойка и её симметрия, подсудность, срок оплаты, форс-мажор, персональные данные, односторонний отказ, приёмка...) → отчёт: ✔ найдено / ⚠ проверьте / ✖ отсутствует, каждая строка с пояснением «почему это важно». Финал: «Настоящий Юрист-агент проверяет по ВАШЕМУ чек-листу и с полным пониманием текста — покажем на демо». Работает без всяких загрузок, мгновенно — главный вау-тест для директора.

### G4. Секретарь — голосовой тест
Кнопка-микрофон: Web Speech API (webkitSpeechRecognition, ru-RU) распознаёт речь посетителя, агент отвечает текстом + голосом (speechSynthesis, русский голос). Сценарии: «запиши меня на завтра», «сколько стоит», «расскажи про компанию». Если API недоступен — честное «в этом браузере голос недоступен, вот текстовый чат». Индикатор записи — пульсирующая красная точка.

### G5. Приложение «Фабрика контента» (наш продукт Content Factory)
Отдельное приложение (иконка из системных, напр. Freeform занят — взять Pages/TextEdit-стиль или Photos-стиль свободную): 
- Визуал конвейера: Тема → Черновик → Редактура → Публикация (анимированная линия с бегущим импульсом).
- Живой генератор: выбрать формат (пост для соцсетей / карточка товара / ответ на отзыв клиента), ввести тему/товар/отзыв, тон (деловой/дружелюбный) → кнопка «Сгенерировать»: уровень WebLLM — настоящая генерация; фолбэк — умные шаблоны с подстановками и вариативностью (3 заготовки на формат×тон, рандомизация оборотов), результат печатается посимвольно, кнопка «Скопировать».
- Продающий блок: «Фабрика включается к вам в контур: календарь публикаций, ваш тон, ваши продукты — от 50 000 ₽/мес» + чип «Записаться».

### G6. Sandbox-бейджи
Везде, где посетитель может «потрогать» (Сообщения, Юрист, Документовед, Секретарь, Фабрика, Калькулятор, 3D-офис, Игра) — единый бейдж «▶ Можно попробовать» на иконках springboard/дока НЕ вешать (шум), а вот в App Store и окне «ИИ-сотрудники» каждая карточка получает кнопку «Попробовать сейчас» → открывает соответствующий тест. На README добавить строку-чип: «Всё на этом сайте можно потрогать: чат, проверка договора, голос, калькулятор →» (ведёт в App Store).
